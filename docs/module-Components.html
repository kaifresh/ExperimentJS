Module: Components

Set some instructions text for your participants. This text will be displayed on every trial.
If you wish to edit the CSS of the instructions <div>, it can be accessed by the ID "#%%INSTRUCTIONS%%"
Parameters:
Name Type Description
instructions_text string instructions to be displayed to particpants on every trial
