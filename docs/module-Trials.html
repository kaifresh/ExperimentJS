Module: Trials

`setIVLevels` AND `setIVsetFunc` must both be called once per IV. They are linked by a common `ivname`.\n Call `setIVLevels` to assign data to the setter function (passed to `setIVsetFunc`). Data should be an array of arrays. Each sub array is the arguments passed to the setter function:
Parameters:
Name Type Description
ivname string The name of the IV.
levels array Array of arrays of arguments passed to the set function.

Example
ExperimentJS.Trials.setIVsetFunc("iv name", function(arg1, arg2, arg3){ ... });
ExperimentJS.Trials.setIVLevels("iv name", [ [arg1_a, arg2_a, arg3_a], [arg1_b, arg2_b, arg3_b], ...]);
