(static) setIVPhases(phase_num, array_of_iv_names, transition_func_or_delay)

Present IVs sequentially by assigning each independent variable (IV) name to an ordinally numbered phase. To transition between phases, a function or a delay (in milliseconds) must be provided. If a function is provided, it must receive a callback as its first argument. This callback should be called to transition to the next phase.
Parameters:
Name Type Description
phase_num int ordinal number of the phase
array_of_iv_names array array of IV names for this phase
transition_func_or_delay function | number function to handle transition between phases, or duration until next phase
Source:
  • core/Trials.js, line 214
Example
// Using a function
ExperimentJS.Trials.setIVPhases(0, ["Name of an IV to run in phase 0", "Another one"], function(continue_to_next_phase){
     if(some_condition_is_satisified){
         continue_to_next_phase();
     }
});

// Using a number (i.e. a timeout delay)
ExperimentJS.Trials.setIVPhases(0, ["Name of an IV to run in phase 0", "Another one"], 2000); // after two seconds go to the next phase
ExperimentJS.Trials.setIVPhases(1, ["An IV in the second phase"]); // after two seconds go to the next phase

(static) setIVResponseParserFunc(ivname, parserFunc) → {string|object}

In some cases, the arguments that you pass into your independent variable (IV) setter function will not be useful to include in the output at the end of the experiment. `setIVResponseParserFunc` allows you to set a function that parses the arguments passed to the setter function into one (or more) output values (with more descriptive value). See example for more detail on output..
Parameters:
Name Type Description
ivname string The name of the independent variable (IV).
parserFunc function Function to parse responses. Must conform to the interface in the example..
Source:

Returns:
One (string) or more (object) columns of parsed output data
Type
string | object
Example
Trials.setIVResponseParserFunc("iv name",
     function ( r, g, b..., index){
     // Here R, G & B are the arguments passed to the setter function 

         // Parse the arguments for this independent variable...

         // eg. three numbers representing an RGB triplet could be condensed into a color name:
         var output;
         if (r === 255 && g === 0 && b === 0){
             output = "red";
         } else {
             output = "not red";
         }

         // return either a:
                             string -    output a single column of parsed data, in string form
                             object -    output multiple columns of parsed data, where each key is the column header

     }
);

(static) setIVsetFunc(ivname, setFunc)

Generating every IV requires 2 steps: creating the levels and creating a set function. `setIVsetFunc` is called to set the function that will be used to set that IV in the display. The levels it receives are set by `setIVLevels`. The arguments this function receives must correspond to the elements in the sub-arrays passed to `setIVLevels`.
